# Python Fundamentals

Este repositório reúne meus estudos em Python, documentando minha evolução desde os primeiros conceitos da linguagem até conteúdos mais avançados voltados para a área de Engenharia de Dados.

O principal objetivo é registrar meu progresso por meio de exercícios, desafios práticos, anotações e projetos, criando um histórico do meu desenvolvimento como programador e um material de consulta para acompanhar minha evolução ao longo do aprendizado.

## Objetivos

- Desenvolver lógica de programação;
- Aprender os fundamentos da linguagem Python;
- Praticar a resolução de problemas por meio de exercícios;
- Construir uma base sólida para Engenharia de Dados;
- Organizar e documentar minha evolução durante os estudos.

## Conteúdos estudados

Atualmente, este repositório contempla estudos relacionados a

- Variáveis;
- Tipos de dados (`int`, `float` e `str`);
- Entrada e coleta de dados com `input()`;
- Conversão de tipos;
- Operadores aritméticos;
- Métodos básicos da linguagem;
- Resolução de problemas utilizando lógica de programação;
- Cálculo de médias, porcentagens e descontos.

Novos conteúdos, exercícios e projetos serão adicionados conforme meu progresso nos estudos.
