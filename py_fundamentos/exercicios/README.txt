# Exercícios

Esta pasta reúne todos os exercícios práticos desenvolvidos durante meu aprendizado em Python.

O objetivo é consolidar os conceitos estudados por meio da resolução de problemas, aplicando cada novo conteúdo em situações práticas. Os exercícios acompanham minha evolução na linguagem e servem como registro do desenvolvimento da minha lógica de programação.

## Organização

Os exercícios são organizados em pastas de acordo com o período ou módulo de estudo.

Cada conjunto de exercícios contém:

- **Enunciados:** arquivo com a descrição de cada desafio proposto;
- **Soluções:** implementações desenvolvidas em Python para resolver cada exercício.

Essa organização facilita a consulta dos problemas e permite comparar os enunciados com suas respectivas soluções.

## Objetivos

- Praticar os conceitos aprendidos em cada etapa dos estudos;
- Desenvolver raciocínio lógico e capacidade de resolução de problemas;
- Criar o hábito de escrever código de forma organizada e legível;
- Documentar minha evolução na linguagem Python.
